Estrutura básica

voltar

Começa o aqurivo com as tags de abertura e fechamento "HTML" e dentro delas as tags "HEAD" e "BODY".

As de aberturas ficam entre "< >" e as de fechamento com uma "/" depois de "<".

Alumas TAGS não tem TAGS de fechamento e algumas possuem apenas a de fechamento.

Tags

voltar

Existem vários tipos de TAGS, alguns exemplos são:

  • Title = titúlo que aparece na aba do navegador
  • I = itálico
  • Strong = negrito
  • A (apenas a de fechamento) = links
  • IMG = inserir imagens
  • Imput = Digitalização de texto, números, cores e etc
  • etc

Atributos

voltar

Nada mas é do que as proprieadades que as TAGS podem ter, podendo ser globais, genéricos ou específicos.

Eles permitem mudar as cores, tamanhos, ações e etc.

Listas

voltar

Existem as listas ordenadas "ol" e as listas não ordenadas "ul".

As duas precisam das TAGS filhas "li" para funcionar, assim tendo que colocar um "li" sempre que for adicionar um item.

Links

voltar

Os links sáo utilizados para navegar no próprio site ou diferentes sites

Como ja mencionado, a TAG para links é a "a" com o "href" seguido do link dentro da mesma

Para abrir links em outra aba, utilizasse o atributo "_blank"

Se quiser usar links para navegar na própria página ultizasse o href com uma hashtag e um id de identificação na parte que deseja ir
